# Template Project

This repository is a neutral Next.js starter you can customize freely.

## Quick Start

1. Install dependencies:
   ```
   npm install
   ```
2. Start the dev server:
   ```
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000)

## Where to Edit

- Main app UI: `src/app/page.tsx`
- Document metadata and root layout: `src/app/layout.tsx`
- Global styles: `src/app/globals.css`

## Tech Stack

- [Next.js 14](https://nextjs.org/)
- [React 18](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
